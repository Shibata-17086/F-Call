# VOICEVOX連携セットアップガイド

F-CallシステムにVOICEVOXの音声合成を統合する手順です。

## 1. VOICEVOXのインストール

### macOS版のダウンロード
1. [VOICEVOX公式サイト](https://voicevox.hiroshiba.jp/)にアクセス
2. macOS版をダウンロード
3. アプリケーションフォルダにインストール

### VOICEVOXの起動
```bash
# VOICEVOXアプリを起動
open -a VOICEVOX
```

## 2. VOICEVOXエンジンAPIの起動

VOICEVOXアプリを起動すると、自動的にローカルAPIサーバーが起動します：
- **URL**: `http://localhost:50021`
- **ドキュメント**: `http://localhost:50021/docs`

### APIの確認
```bash
# APIが起動しているか確認
curl http://localhost:50021/speakers
```

## 4. F-Callでの設定

### 管理画面での設定
1. 管理画面の「音声設定」セクションで「VOICEVOX使用」を有効化
2. キャラクター（speaker_id）を選択
3. 速度・ピッチを調整

### 推奨設定（男性風の声）
- **キャラクター**: 雨晴はう（speaker_id: 10）
- **速度**: 1.1
- **ピッチ**: -3（低め）

## 5. トラブルシューティング

### VOICEVOXに接続できない
```bash
# VOICEVOXが起動しているか確認
curl http://localhost:50021/speakers

# エラーの場合、VOICEVOXアプリを再起動
```

### 音声が再生されない
- ブラウザの音声再生が許可されているか確認
- VOICEVOXエンジンのログを確認
